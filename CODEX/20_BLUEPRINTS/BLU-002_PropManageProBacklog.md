> **BLUF:** This document contains the 7-task backlog generated by the DarkGravity Architecture swarm, decomposing the PropManage Pro blueprint (BLU-001) into atomic, executable engineering tasks.

## Backlog

### 1. Initialize Project Structure and Core Dependencies
Create the root project structure and install core dependencies for a Next.js 14/15 (App Router) application utilizing Tailwind CSS, shadcn/ui, TanStack Query v5, and Lucide-React.

### 2. Configure Firebase Project and Initialize SDK
Set up the Firebase project in the console (Firestore, Auth, Storage) and create the initialization logic in the application.

### 3. Implement Core Data Models and Firestore Schema
Define TypeScript interfaces for the core Firestore collections (`User`, `Property`, `Tenant`, `Ledger`).

### 4. Build Property CRUD API Layer and Hooks
Create the data access layer for the `properties` collection. Implement functions for Create, Read, Update, and Delete operations.

### 5. Develop Core Dashboard KPI Component
Build a reusable KPI card component that displays a single metric (e.g., Total Revenue, Occupancy Rate).

### 6. Implement Basic Firestore Security Rules
Write and deploy foundational Firestore security rules to enforce data isolation (Multi-Tenancy based on `owner_uid`).

### 7. Create Authentication Context and Protected Route
Set up a React context to manage Firebase authentication state (user, loading) and protect dashboard routes from anonymous access.

### 8. Implement Data Aggregation Cloud Functions (KPI Dashboard)
Create Firebase Cloud Functions triggered by `ledgers` writes to compute and update a centralized `portfolio_stats` document, reducing dashboard load to 1-read (GCP Free Tier optimization).

### 9. Build Client-Side Media Compression Utility
Implement a `browser-image-compression` utility within client forms to ensure all tenant attachments (e.g. maintenance photos) are scaled down to under 500KB prior to GCP Cloud Storage upload.

### 10. Scaffold Advanced Enterprise Modules
Lay down the foundation and UX scaffolding for the missing enterprise workflows: E-Signature Integration, Vendor/Work Order Management, and Application Screening Pipelines.
